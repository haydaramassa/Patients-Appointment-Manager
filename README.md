# Patient Appointment Manager

A desktop clinic management system built with **JavaFX** for the frontend and **Spring Boot** for the backend.

## Project Status
This project is currently **under active development** and is **not yet a final release**.  
The current version already includes working core modules, while additional features, UI improvements, and refinements are still in progress.

## Current Features
- User authentication
- Role-based access (`ADMIN`, `SECRETARY`)
- Dashboard summary
- Patients management
- Appointments management
- Users management for admin
- JavaFX desktop frontend connected to backend APIs

## Tech Stack
- Java 25
- JavaFX
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL / MariaDB
- Maven

## Project Structure
- `patient-appointment-backend`
- `patient-appointment-frontend`

## Notes
- The project is functional in its current state
- UI and structure are still being improved
- More modules and refinements will be added gradually
